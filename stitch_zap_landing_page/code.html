Why I Built This

This is a portfolio project inspired by tools like Jira. Built to learn real-world backend architecture and system design. Focus on practical engineering, not just UI.

The goal was to tackle complex relational data modeling, secure file handling, and granular permission logic while maintaining a developer-first interface that remains snappy under heavy load.
